# Productify

A modern e-commerce platform for listing and managing earbuds products with social interaction features.

## Overview

Productify is an e-commerce website that allows users to list, manage, and interact with earbuds products. Users can create accounts, showcase their products, and engage with the community through comments and interactions.

## Features

### Authentication
- **Sign Up**: Create a new account to start listing products
- **Sign In**: Access your account and manage your listings

### Product Management
- **List Products**: Add your earbuds to the platform with details and images
- **View Products**: Browse all listed earbuds from yourself and other users
- **Edit Products**: Update your product information anytime
- **Delete Products**: Remove products you no longer want to list

### User Profile
- **Personal Dashboard**: View all your created products in one place
- **Profile Management**: Access and manage your account information

### Social Interaction
- **Comments**: Engage with products by leaving comments
- **Community Engagement**: Interact with your own products and others' listings
- **Product Discussions**: Join conversations about different earbuds

## Getting Started

### Prerequisites
- Node.js (recommended version)
- npm or yarn package manager
- Modern web browser

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd productify
```

2. Install dependencies
```bash
npm install
```

3. Set up environment variables
```bash
cp .env.example .env
```

4. Start the development server
```bash
npm run dev
```

5. Open your browser and navigate to `http://localhost:3000`

## Usage

1. **Create an Account**: Sign up with your email and password
2. **List Your Earbuds**: Navigate to the product creation page and fill in the details
3. **Manage Your Products**: Go to your profile to view, edit, or delete your listings
4. **Explore**: Browse products from other users and leave comments
5. **Engage**: Interact with the community through comments and discussions
